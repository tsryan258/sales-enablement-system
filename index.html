Step 1 — Build Your Google Sheet
Three tabs, one Sheet
1
Create a new Google Sheet Name it something like PMM Enablement OS. Add three tabs: Assets, Tasks, and Feedback.
2
Assets tab — required columns (Row 1) name · type · status · icp · stage · ai_layer · owner · last_updated · notes
3
Status values to use Current · Needs Review · Stale · Draft · Retired
4
AI Layer values to use AI-Generated · AI-Assisted · Human-Only
5
last_updated format Use MM/DD/YYYY format for freshness scoring to work correctly.
Step 2 — Publish Your Sheet
Get the CSV URL for each tab
1
File → Share → Publish to web In your Google Sheet, go to File → Share → Publish to web.
2
Select the Assets tab, CSV format In the dropdown, choose the Assets tab and select Comma-separated values (.csv), then click Publish.
3
Copy the URL The URL looks like: https://docs.google.com/spreadsheets/d/[ID]/pub?gid=[GID]&single=true&output=csv
4
Paste into Asset Registry tab Go to the Asset Registry tab in this app and paste the URL into the connection field.
5
Repeat for Feedback tab Publish the Feedback tab the same way and paste that URL in the Feedback section of Setup below.
Step 3 — Build Your Google Form
Sales feedback intake, no login required
1
Create a Google Form Go to forms.google.com and create a new form titled "Enablement Feedback."
2
Suggested fields Your name · Asset name (dropdown) · Issue type (Not working / Outdated / Missing / Other) · Notes / details
3
Link to your Sheet In the Form, click Responses → Link to Sheets → select your existing PMM Enablement OS sheet. Responses will land in a new tab automatically.
4
Get the embed URL Click Send → Embed (< > icon) → copy the src URL from the iframe code. Paste it in the Sales Feedback tab.
